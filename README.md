            WELCOME TO PARADOX HOTEL (v. 404.500)
=========================================================

[ENG] - ENGLISH
---------------------------------------------------------
STORY
"Welcome to Paradox Hotel" is not a physical place. It is a sentient 
digital architecture, a recursive reality management software that 
has stopped responding to external commands. The player controls 
SAMUEL, a 404 Error (Not Found). Having materialized in Room 404, 
Samuel is a rogue fragment of code wandering the system in search 
of an exit that logic does not provide.

SYSTEM AGENTS
* THE CONCIERGE (Kernel): The primary interface. Manages access.
* THE ELEGANT WOMAN (Redirect/Cache): Redundant data stuck in the lounge.
* THE MAN IN 202 (Validation Process/OK Gate): The verification 
  process. Without his formal "OK," no data packet can be forwarded. 
  He is the system's approval filter.
* THE TWINS (Duplicate Process/Race Condition): Two identical instances 
  creating a logical loop, stalling progression in the hallway.
* THE MAID (Memory Cleaner/Access Control): Manages room cleanup 
  and restricts access to critical sectors like Suite 500.
* THE CHEF (Security Audit): An aggressive process to delete bugs.
* THE MAINTENANCE WORKER (Background Process): Follows absolute instructions.

=========================================================
          TECHNICAL STACK & DEVELOPMENT NOTES
=========================================================

CORE FRAMEWORK: Angular 21.0.0 (Cutting Edge)
- REACTIVE ENGINE: Angular Signals for real-time inventory and 
  game-state synchronization (No Zone.js overhead).
- STYLING: Tailwind CSS 4.0 + @angular/material 21.
- ANIMATIONS: Motion (Framer Motion for Web) for seamless room 
  transitions and terminal-style UI effects.
- ARCHITECTURE: Component-based design with dedicated Audio 
  Crossfade Service and Custom i18n Mapping.

AI COLLABORATION: Gemini 3 Flash
- LEAD LOGIC ARCHITECT: Implementation of the "Bootstrap Paradox" 
  game loop and complex binary fragment validation.
- PROMPT ENGINEERING: Advanced persona-driven dialogue 
  generation for a consistent Noir/Kafkian atmosphere.

SOUNDTRACK: SUNO AI
- GENRE: Noir Jazz & Industrial Drone.
- DYNAMIC AUDIO: Managed via AudioService with a 2-second 
  linear crossfade between 4 main recursive tracks.
- FILING: Assets served locally via /assets/audio/ path.

DEPLOYMENT & ENVIRONMENT
- INFRASTRUCTURE: Vercel (Production) / GitHub (Source Control).
- BUILD TOOL: Angular CLI with SSR (Server-Side Rendering) support.
- API INTEGRATION: Pre-configured for @google/genai 1.27.0.
